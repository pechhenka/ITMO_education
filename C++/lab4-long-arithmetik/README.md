# Длинная арифметика
Реализация класса LN (длинная арифметика) для целых чисел с произвольной точностью и со знаком, которые могут быть NaN. 

Поддерживаются следующие операции:
* \- вычитание
* \+ сложение
* / деление
* \* умножение
* % остаток от деления
* sqrt корень
* операции сравнения
* операция перевода в строку
